Introduction

opacityChangeOnScroll as the name suggests, is a Javascript library that makes it easy to alter the opacity of an element on the page using the scroll event.

Getting Started

Dependencies

jQuery

                            
                                <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js" type>

Loading the Library

                            
                                <script src="https://cdn.rawgit.com/sukritchhabra/opacityChangeOnScroll/master/opacityChangeOnScroll.js" type>

Options

increase-on-scroll-down

Increases the opacity of the element as you scroll down.

                            
                                <img class="increase-on-scroll-down" src="img1.png" alt>

decrease-on-scroll-down

Decreases the opacity of the element as you scroll down.

                            
                                <img class="decrease-on-scroll-down" src="img1.png" alt>

increase-decrease-on-scroll-down

Increase First, then Decrease opacity of the element as you scroll down

                            
                                <img class="increase-decrease-on-scroll-down" src="img1.png" alt>

decrease-increase-on-scroll-down

Decrease First, then Increase opacity of the element as you scroll down

                            
                                <img class="decrease-increase-on-scroll-down" src="img1.png" alt>

Custom Attributes

data-start

Use this to define where the change/animation begins.

Value assigned to this attribute specifies the percentage of the windows height, from the bottom of the window where the change begins.

                            
                                <img class="increase-on-scroll-down" data-start="50" src="img1.png" alt>

data-end

Use this to define where the change/animation ends.

Value assigned to this attribute specifies the percentage of the windows height, from the top of the window where the change ends.

                            
                                <img class="decrease-on-scroll-down" data-end="30" src="img1.png" alt>

Use Together

Any of the above attributs above can be used together.

                            
                                <img class="increase-on-scroll-down" data-start="40" data-end="30" src="img1.png" alt>
