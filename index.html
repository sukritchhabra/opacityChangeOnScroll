Introduction

opacityChangeOnScroll as the name suggests, is a Javascript library that makes it easy to alter the opacity of an element on the page using the scroll event.

Options

increase-on-scroll-down

Increases the opacity of the element as you scroll down.

                        
                            <img class='increase-on-scroll-down' src='img1.png'>

increase-on-scroll-down

Decreases the opacity of the element as you scroll down.

                        
                            <img class='decrease-on-scroll-down' src='img1.png'>

increase-on-scroll-down

Increase First, then Decrease opacity of the element as you scroll down

                        
                            <img class='increase-on-scroll-down' src='img1.png'>
